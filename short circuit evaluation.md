
وقتی از and استفاده میکنیم، مقداری که برابر True هست برگشت داده میشود. اگر کل عبارت ترکیبی از چندین مقدار True باشد آخرین مقدار به عنوان نتیجه برگشت داده میشود. 
اگر عبارت ترکیبی از مقادیر False باشد اولین مقدار False برگشت داده می شود.
اگر عبارت ترکیبی از مقادیر True و False باشد اولین مقدار False برگشت داده می شود.

```python
>>> False and True        # => False
>>> 1 and 2               # => 2
>>> 1 and 2 and 3 and 4   # => 4
>>> 1 and 0 and '' and 2  # => 0
>>> 0 and [] and ''       # => 0
```


وقتی از or استفاده می کنیم اولین مقدار True در سمت چپ برگشت داده می شود. هر کجا یک عبارت True باشد برگشت میخورد.
اگر همه عبارت ها True باشند اولین مقدار True برگشت داده  میشود.
اگر همه عبارت ها False باشد آخرین مقدار False برگشت داده میشود.
اگر ترکیبی از مقادیر True و False داشته باشیم اولین مقدار True برگشت داده میشود.

```python
>>> False or True       # => True
>>> 1 or 2              # => 1
>>> 1 or 2 or 3         # => 1
>>> 1 or 0 or '' or 2   # => 2
>>> 1 or 0 or ''        # => 1
>>> 0 or [] or ''       # ''
```

من در مثالهای بالا روی نتیجه برگشتی تمرکز کرده ام. باید به این نکته توجه کنیم که بررسی از سمت چپ به راست انجام می‌شود. در and تا هنگامی که به یک مقدار False نرسیده عبارتها اجرا می شوند. اگر همه عبارتها True باشند آخرین عبارت True برگشت داده میشود. هر کجا به False رسیدم آن عبارت False برگشت داده میشود.[]
